# MSCFE690_Capstone
This project studies how the weather index information will affect the commodity price predictions in long term data using FB Prophet model. 

The required packages are outlined below: 
Seaborn 0.11.2
Prophet 1.0.1
Scikit-learn 1.1.3
Pandas 1.4.2
Statsmodels 0.13.2
Numpy 1.24.4
Matplotlib 3.6.3
